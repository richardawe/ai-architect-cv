CV Generator

Paste a job description to generate a tailored CV via GitHub Actions — the same LLM setup as the EU261 project. This page triggers the generate-cv.yml workflow in your repo.

1 Create a GitHub Personal Access Token at github.com/settings/tokens with the workflow scope (classic token).
2 Make sure OPENROUTER_API_KEY is set as a repository secret in Settings → Secrets.
3 Paste your token below. It stays in sessionStorage — it never leaves your browser.
